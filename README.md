# BeatTheSystem

#How to install:
* Install GitKraken or similar Github application for windows
* Clone the BeatTheSystem repo into a local folder. Now you can track the commits/changes to the code in this repo that are made over time.
* Have the latest version of python installed
* Open up the repo folder with Visual Studio Code (free) and start the terminal/console. Or open command prompt\powershell console at the repo folder.
* Create a virtualenv by typing
`py -m venv env`
Into the console. This creates a virtual python environment which will be a project specific environment.
* This is a clean environment that keeps the modules needed for this project separate from the modules in other projects.
* Once the virtualenv is created, you can start the virtualenv by entering
`.\env\Scripts\activate`
into to the console.
* Now that the virtualenv is started, you can install the required packages by typing
`pip install -r requirements.txt`
* Now your virtualenv should be ready to run the project code!
